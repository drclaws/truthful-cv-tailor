# Template Selection Rules

Use `cv-ats-agent-template.tex` by default.

The default template is the compact columnar production template:
- main column for narrative evidence such as Experience, Projects, Education;
- signal column for Summary, Skills, Languages, and compact supported signals;
- full-width contact header with visual icons and visible text labels.

Use a simplified render when the column extraction gate fails, the target system
requires a stricter format, or the content does not fit the compact composition
without harming readability.

Do not select, rewrite, or restyle a template merely to squeeze content onto the
page. For a one-page target, first optimize validated Experience content
moderately. If later content edits create extra space, reuse it for the most
important supported Experience information before considering any template
fallback.
